CSS Selectors

CSS selectors let you identify or select specific HTML elements. A couple of references can be found at W3 schools, the documentation and at this page. The three most popular include id, class, and element (i.e., type).

Whatever selector you choose, the workflow will be the same.

  1. Select an element to style.
  2. Create an object.
  3. Write the CSS rule for that elementâ€™s property you wish to style.

selector { property: value; }

Most frequently used CSS selectors
Selector Example Description
#id #firstName Selects the element with id="firstName"
.class .intro Selects all elements with class="intro"
element p Selects all <p> elements

Combinator Selectors

When you need to be more specific with your selectors, you can combine them. There are four different combinators in CSS:

Combinator Selectors
Selector Example Description
.class1.class2 .className1.className2 Selects all elements with both className1 and className2
.class1 .class2 .className1 .className2 Selects all elements with className2 that is a descendant of an element with className1
element, element div, p Selects all <div> elements and all <p> elements
element + element div + p Selects the first <p> element that is immediately after <div> elements
element > element div > p Selects all <p> elements where the parent is a <div> element
element ~ element ul ~ p Selects any <p> elements that follow a <ul> element

Attribute Selectors

An attribute selector matches elements based on the presence or value of a given attribute.

Attribute Selectors
Selector Example Description
element[attr] a[title] Selects the anchor tags that have a title attribute

Pseudo-classes

A pseudo-class is a keyword added to a selector that specifies a special state of the selected element(s), such as :hover, :focus, :active, :visited.

The syntax of a pseudo-class is as follows:
selector:pseudo-class{
  property: value;
}

Pseudo-classes
Selector Example Description
:pseudo-class a:active Selects the active link
